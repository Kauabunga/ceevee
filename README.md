# Carson Bruce CV

## Get started

1. `npm install`
2. `npm run dev`

## Deploy

Deployment for a root github page is done via the repository https://github.com/Kauabunga/kauabunga.github.io

1. `npm run build`
2. `npm start`
3. `npm run deploy:clone`
4. `npm run deploy`
5. `cd kauabunga.github.io && git commit -am "Updated deployment" && git push`
